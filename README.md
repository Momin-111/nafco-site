# NAFCO Website

Static bilingual (English / Arabic) website for NAFCO — National Aluminium & Fabrication Company.

## Structure

```
├── index.html            # Home (English)
├── index-ar.html         # Home (Arabic)
├── about.html / about-ar.html
├── products.html / products-ar.html
├── projects.html / projects-ar.html
├── certificates.html / certificates-ar.html
├── contact.html / contact-ar.html
├── css/                  # Stylesheets
├── js/                   # Scripts (animations)
├── images/               # Photos, logos, partners, projects
├── videos/               # Hero & testing videos
└── documents/            # ISO certificate & audit report PDFs
```

## Running locally

Open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 8000
```

then visit http://localhost:8000
